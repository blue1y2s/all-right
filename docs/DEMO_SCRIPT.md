# Demo 脚本

建议 2 分钟内完成。先演示产品闭环，再讲扩展架构。

## 1. 工作台输入（0:00-0:20）

打开首页，展示左侧统一源内容：标题、Markdown 正文、语气、封面、目标读者和标签。

## 2. 平台适配（0:20-0:50）

点击公众号、知乎、B站、小红书卡片，展示每个平台会生成不同标题、正文结构、标签和约束提示。

## 3. 热点输入与 AI 增强（0:50-1:10）

在“热点输入”中输入一个主题，点击任意选题导入大纲。说明无 API Key 时使用规则模板，有 Key 时可启用 AI 改写增强。

## 4. 草稿复制和导出（1:10-1:25）

在预览右侧点击“复制”或“导出”，说明运营同学可以把单个平台草稿带走，也可以下载 JSON 草稿包。

## 5. 一键模拟发布和撤回（1:25-1:45）

点击“一键模拟发布”，展示四个平台进入队列、日志流转和历史记录。成功后点击“模拟撤回”，说明真实平台会由 adapter 判断是否支持撤回、下架或删除。

## 6. 扩展架构（1:45-2:00）

切到“扩展架构”，说明新增平台只要新增 adapter；真实发布可接官方 API、Wechatsync CLI/MCP 或 Playwright。
